PASONAの法則とは？

PASONAの法則は、日本のマーケティングコンサルタント神田昌典氏が提唱した、セールスライティング・提案構成のフレームワークです。LP（ランディングページ）やDM、営業メール、プレゼン資料など幅広い場面で活用されています。

最大の特徴は「顧客の感情の流れに沿ったストーリー設計」にあります。課題を認識させ、感情を動かし、論理的な解決策を示し、最後に行動を促す――この流れを5ステップで再現性高く構築できるのが強みです。

PASONAの5ステップ

各ステップの頭文字を取って「PASONA」。顧客の心理変化に沿った順序で提案を組み立てます。

P
Problem

問題提起

顧客が直面している課題や悩みを言語化し、「それ、まさに自分のことだ」と感じさせるステップです。ターゲットの具体的な状況を描写し、共感を得ることがゴールです。

スクリプト例

「新人が入社しても戦力化するまで半年以上かかる、スクリプトが属人化して品質にばらつきがある、こうしたお悩みはありませんか？」

A
Agitation

問題の深掘り

問題を放置するとどうなるかを具体的に示し、「このままではまずい」という危機感を醸成します。新PASONAでは「煽り」ではなく「親近感と共感」がキーワードです。

スクリプト例

「属人化が進むほど、エース社員の退職リスクが組織全体のリスクになります。実際に、スクリプトの引継ぎができず売上が3割落ちた事例もあります。」

SO
Solution

解決策の提示

課題を解決する方法を提案します。ここで初めて自社の商品・サービスの説明に入りますが、機能の羅列ではなく「どう課題を解決するか」のストーリーで語ります。

スクリプト例

「営業スクリプトAIは、業種・ターゲット・目的に最適化されたスクリプトをAIが即座に生成。誰でもトップ営業品質のスクリプトをすぐに使えます。」

N
Narrowing

絞り込み

対象を絞り込み、限定感や緊急性を演出するステップです。「今だけ」「先着○社」「○○業界限定」など、行動を先延ばしにさせない仕掛けを入れます。

スクリプト例

「現在、無料トライアルは1日2回までご利用いただけます。今なら初期設定のサポートも無料で対応しています。」

A
Action

行動喚起

最後に、顧客が取るべき具体的なアクションを明確に伝えます。「検討してみてください」ではなく、「今すぐ○○してください」と明確な指示にすることがポイントです。

スクリプト例

「まずは無料で1本スクリプトを生成して、チームで試してみてください。3分で完了します。」

Before / After：PASONAで変わる提案

同じ商材でもPASONAの構成を使うだけで、提案の説得力が大きく変わります。

✗ Before（よくある提案）

弊社は営業スクリプト自動生成AIを提供しています。業種別に最適化されたスクリプトを生成でき、電話・メール両方に対応しています。月額980円からご利用いただけます。ぜひご検討ください。

✓ After（PASONA構成）

営業スクリプトの品質が属人化していませんか？（P）エース社員の退職で売上が急落するリスクは年々高まっています。（A）AIが業種×ターゲットに最適化されたスクリプトを即時生成。全員が同品質で営業できます。（SO）今なら無料トライアル実施中。（N）まずは1本、3分で試してみてください。（A）

PASONA実践の5つのコツ

💡 実践ポイント

  • Pは「相手の言葉」で語る：自分の言葉ではなく、ターゲットが実際に使う表現で問題提起しましょう。事前ヒアリングやアンケートから言葉を拾うと刺さりやすくなります。
  • Aは煽りすぎない（新PASONA）：旧PASONAの「Agitation=煽り」から「Affinity=親近感」に進化しています。恐怖ではなく共感ベースで課題を深掘りする方が信頼を得られます。
  • SOでは「変化」を見せる：機能一覧ではなく、導入前後でどう変わるかを見せましょう。BeforeAfterのビジュアルやストーリーが最も効果的です。
  • Nは嘘をつかない：「先着10社」が実は無制限だった…となると信頼を失います。限定する理由に合理性があるか、常にチェックしましょう。
  • Aは1アクションに絞る：「資料請求も無料相談もお問い合わせも」と選択肢を増やすと逆に行動率が下がります。最も取ってほしい1つのアクションだけを明確に提示しましょう。
